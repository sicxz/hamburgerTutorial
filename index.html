CSS-Only Hamburger Menu Tutorial

Learn how to create a responsive hamburger navigation menu using only HTML and CSS—no JavaScript required!

Why no JavaScript? While JavaScript can make interactive elements more powerful, CSS-only solutions are:
  • Faster to load and execute
  • Less prone to bugs and compatibility issues
  • More accessible when JavaScript is disabled
  • Often simpler to implement and maintain

This tutorial will teach you how to create a hamburger menu that transforms into an "X" when opened, with a smooth sliding animation—all without writing a single line of JavaScript.

See It In Action

This website itself uses the CSS-only hamburger menu we're teaching! Try resizing your browser window to under 768px width, or view on a mobile device to see it transform into a hamburger menu.

Click the hamburger icon to open the menu and see how it smoothly slides in and transforms.

Learn How To Build It

Step-by-Step Tutorial

Let's build a hamburger menu from scratch using only HTML and CSS:

Set up the HTML Structure

First, let's create the basic HTML structure for our page:

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS-Only Hamburger Menu</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <div class="nav-container">
            <div class="logo">MyWebsite</div>
            
            <!-- Navigation will go here -->
            
        </div>
    </header>
    
    <main>
        <h1>CSS-Only Hamburger Menu</h1>
        <p>Resize the browser to see the hamburger menu in action.</p>
    </main>
</body>
</html>

Add the Navigation Elements

Now, let's add the key components of our hamburger menu:

HTML 
<header>
    <div class="nav-container">
        <div class="logo">MyWebsite</div>
        
        <!-- The checkbox hack - this is the core of our JavaScript-free solution -->
        <input type="checkbox" id="menu-toggle">
        
        <!-- Hamburger icon as a label -->
        <label for="menu-toggle" class="menu-button">
            <span class="hamburger"></span>
        </label>
        
        <!-- Navigation menu -->
        <ul class="nav-menu">
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
            <li><a href="#">Services</a></li>
            <li><a href="#">Portfolio</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </div>
</header>

The key elements are:

  • A hidden checkbox (#menu-toggle)
  • A label with the hamburger icon that will toggle the checkbox
  • The navigation menu that will be shown/hidden

Style the Base Elements

Let's create the basic CSS styles:

CSS 
/* Reset and base styles */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

/* Header and navigation container */
header {
    background-color: #333;
    color: white;
    padding: 1rem;
    position: relative;
}

.nav-container {
    max-width: 1200px;
    margin: 0 auto;
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.logo {
    font-size: 1.5rem;
    font-weight: bold;
}

/* Hide the checkbox */
#menu-toggle {
    display: none;
}

Create the Hamburger Icon

Now, let's create the hamburger icon using CSS:

CSS 
/* Styling for the hamburger icon */
.menu-button {
    display: none; /* Hidden on desktop */
    cursor: pointer;
}

.hamburger {
    display: block;
    width: 30px;
    height: 3px;
    background-color: white;
    position: relative;
    transition: all 0.3s ease;
}

/* Creating the top and bottom lines of the hamburger */
.hamburger::before,
.hamburger::after {
    content: "";
    display: block;
    width: 30px;
    height: 3px;
    background-color: white;
    position: absolute;
    transition: all 0.3s ease;
}

.hamburger::before {
    top: -8px;
}

.hamburger::after {
    bottom: -8px;
}

Here, we're using a single span element with ::before and ::after pseudo-elements to create the three lines of the hamburger icon.

Style the Navigation Menu

Let's style our navigation menu for the desktop view:

CSS 
/* Navigation menu styles */
.nav-menu {
    display: flex;
    list-style: none;
}

.nav-menu li {
    margin-left: 1.5rem;
}

.nav-menu a {
    color: white;
    text-decoration: none;
    transition: color 0.3s ease;
}

.nav-menu a:hover {
    color: #ff9900;
}

Add the Responsive Styles

Now, let's create the mobile styles using media queries:

CSS 
/* Responsive styles */
@media (max-width: 768px) {
    .menu-button {
        display: block; /* Show hamburger on mobile */
        z-index: 999;
    }
    
    .nav-menu {
        position: fixed;
        top: 0;
        right: -100%; /* Hide off-screen initially */
        width: 250px;
        height: 100vh;
        background-color: #222;
        flex-direction: column;
        padding: 80px 0 30px 0;
        transition: right 0.3s ease;
        z-index: 998;
    }
    
    .nav-menu li {
        margin: 0;
        width: 100%;
        text-align: center;
    }
    
    .nav-menu a {
        display: block;
        padding: 15px;
        border-bottom: 1px solid #444;
    }
}

These styles will transform our horizontal desktop menu into a vertical mobile menu that's positioned off-screen to the right.

Implement the Checkbox Hack

Finally, let's add the CSS that makes the menu toggle work without JavaScript:

CSS 
@media (max-width: 768px) {
    /* Previous mobile styles... */
    
    /* The magic: when checkbox is checked, show the menu */
    #menu-toggle:checked ~ .nav-menu {
        right: 0;
    }
    
    /* Animate hamburger to X when menu is open */
    #menu-toggle:checked ~ .menu-button .hamburger {
        background-color: transparent;
    }
    
    #menu-toggle:checked ~ .menu-button .hamburger::before {
        transform: rotate(45deg);
        top: 0;
    }
    
    #menu-toggle:checked ~ .menu-button .hamburger::after {
        transform: rotate(-45deg);
        bottom: 0;
    }
}

This is the key part of our solution. When the checkbox is checked:

  1. The menu slides in from the right (changing right: -100% to right: 0)
  2. The hamburger icon transforms into an X by:
    • Making the middle line invisible
    • Rotating the top and bottom lines to form an X

Test and Refine

Now it's time to test your menu! Resize your browser window to see how it transforms from a horizontal desktop menu to a hamburger menu on mobile.

You can adjust the following to customize your menu:

  • Menu width: Change the width value of .nav-menu
  • Animation speed: Modify the transition timing
  • Colors: Update the background and text colors
  • Breakpoint: Change the max-width value in the media query

How It Works

Let's break down how this CSS-only hamburger menu works:

The Checkbox Hack

The core of this solution is the "checkbox hack" - a technique that uses a hidden checkbox input element and its :checked state to create toggle functionality without JavaScript.

Key Components

  1. Hidden Checkbox: The #menu-toggle input is hidden with display: none, but it's still functional.
  2. Label as Trigger: The label element is associated with the checkbox via the for attribute, allowing it to toggle the checkbox when clicked.
  3. CSS Selectors: We use the #menu-toggle:checked selector to detect when the checkbox is checked, and the general sibling combinator (~) to target elements that follow the checkbox.
  4. Transitions: CSS transitions create smooth animations for the menu sliding in/out and the hamburger-to-X transformation.

The Magic Selector

The most important part of the code is this selector:

#menu-toggle:checked ~ .nav-menu

This means: "When the checkbox with ID 'menu-toggle' is checked, apply these styles to its sibling element with class 'nav-menu'."
